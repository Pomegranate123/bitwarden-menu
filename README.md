# bitwarden-menu
Yet another bitwarden menu, made using rust & rofi.

## Usage
Run `bitwarden-menu` from the command line, or set a keybind to run it for you.

## Installation
```
$ cargo install --git https://github.com/Pomegranate123/bitwarden-menu.git
```
